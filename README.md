# Patrones de incidentes de aviación en Estados Unidos, 1978–2015

Visualización interactiva de incidentes de aviación registrados por la FAA entre 1978 y 2015.

## Ejecución local

Desde la raíz del proyecto:

```bash
python -m http.server 8000
```

Después abrir:

```text
http://localhost:8000
```

## Tecnologías

- HTML
- CSS
- JavaScript
- Plotly.js
- PapaParse
- GitHub Pages

## Fuente de datos

FAA / ASIAS - Accident and Incident Data System.

## Limitación metodológica

La visualización muestra volumen de incidentes registrados, no tasas de riesgo. El dataset no incluye el número total de vuelos u operaciones por año, estado o fase de vuelo.
